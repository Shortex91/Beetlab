# BeetLab

Der intelligente Weg zu deinem perfekten Beet.

## Vision
BeetLab hilft dir dabei, dein Beet intelligent zu planen, Pflanzen optimal zu kombinieren und den Ertrag zu maximieren.

## Features (coming soon)
- Automatische Beetplanung
- Mischkultur-Optimierung
- Sonnensimulation
- Smarte Empfehlungen
